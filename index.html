Bienvenido a

Descubre al nuevo contendiente en el campo de sistemas operativos Linux.

Veloz

Utiliza un kernel (Zen) optimizado para ser más veloz y responsivo a comparación de su versión genérica

Actualizado

Utiliza una herramienta dedicada para instalar el software más nuevo.

Precavido

Garuda usa el formato de disco BTRFS, que es capaz de auto-repararse y recuperar archivos corruptos.

Seguro

Seguridad incomparable gracias al modelo de software de Código Abierto. Con millones de ojos en el código, Las distribuciones de Linux son los sistemas operativos más seguros que existen.

Instalación Intachable

Garuda ofrece una sencilla instalación para un sistema operativo poderoso con el diseño de UI y UX en mente, El sistema operativo es capaz de ejecutarse desde un USB y ofrece la instalación y gestión automática de controladores con la facilidad de uso en mente. Ya no más software basura del fabricante.

Rendimiento Puro

Garuda cuenta con una herramienta para mantenimiento, limpieza y restauración del sistema con la facilidad de uso en mente. Elimine paquetes no deseados, o reinstale todo el sistema operativo con un par de clicks si lo desea.

Descargar

Garuda cuenta con diez diferentes entornos de escritorio, los cuales tienen sus propias características y flujos de trabajo. Dependiendo si viene de Windows o MacOS, puede que te sientas más familiar con unos que con otros.

Simple pero poderoso. El entorno de escritorio más popular y compatible entre distribuciones.
